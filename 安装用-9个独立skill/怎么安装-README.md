# 怎么安装这 9 个 skill

这里面有 **9 个独立 skill**（00-08），每个都能单独调用。

## 一键安装（推荐）
在这个文件夹里打开终端，跑：
```bash
bash 一键安装.sh
```
它会把 9 个 skill 全部复制进 `~/.claude/skills/`。

## 手动安装
把这里面 9 个文件夹（`00-地基守卫setup` … `08-完整系统串联`）**逐个复制**进 `~/.claude/skills/`。

## 怎么用
- **第一次 / 新业务** → 调 `suma-ai-00-setup`（守门人，会带你建地基 + Notion 骨架）。
- 想单独做某件事 → 调对应的（如 `suma-ai-02-copy` 写文案、`suma-ai-04-video` 做视频）——它会先检查地基做了没。
- 想整条走完 → 调 `suma-ai-08-full`。

🔴 每个 skill 都会：一题一题问（选项卡）· 强制先做 Notion 地基 · 强制记录业务 md · 出的东西套大师+NLP+人性需求。
