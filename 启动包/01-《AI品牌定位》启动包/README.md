**80% 的结果，由最初 20%（方向 / 地基）决定。**

品牌定位 ＝ 你的「**原始资料库**」。喂错了，后面出的每一张图、每一条文案、每一支视频，都是**精美但没用**的东西。

## 🧱 品牌地基 · 7 题（教练会一题一题问你）

> 课前先想好，当天就不卡。写成一个文件存好。

**① 一句话说清你的生意 / 品牌做什么**
> 例：SUMA 是「教人用 AI 做营销内容的课程」。

**② 品牌名 / 主打产品 / CTA 链接**
> 内容最后要把人导去哪里（报名页 / WhatsApp / 官网）。

**③ 目标受众 3–5 个**
> 例（SUMA）：内容创作者、服装店老板娘、美妆美容院老板娘、补习老师…
> 👉 想不出来？让教练帮你 brainstorm。

**④ 每个受众的 persona：气质 / 会在什么场景出现**
> 之后 AI 出的人物就照这个。一两句就行，不用写很细。
> 🔴 人物设定要**贴你当地**（例：SUMA 一律马来西亚本地人）。

**⑤ 你的合规红线：哪些词不能用、不能写什么**
> 例（SUMA）：不写课程价格、不写「免费」、不写具体名额数字（用「名额有限」）、品牌名要写完整。
> 👉 这条是**护栏** —— 写下来，AI 就不会替你闯祸。

**⑥ 你主走哪条路线？**（🔴 核心分叉，决定之后每个 AI Agent 的角度）
| 路线 | 是什么 | 之后的 AI 会怎么跑 |
|---|---|---|
| **① 个人 IP 内容** | 做你自己的影响力 / 人设，靠观点 + 持续输出吸粉、建立信任 | 文案 ＝ 个人 IP 文案 Agent · 影片 ＝ 个人 IP / AI 数字人口播 |
| **② 卖产品 / 服务** | 直接推你的货 / 服务成交 | 文案 ＝ 卖货文案 Agent · 影片 ＝ 产品 / 服务影片广告 |
> 两条都能做，但**每条内容先选定一条**。

**⑦ 成品命名格式**（选一个，之后全部统一用）
| 格式 | 例子 | 好处 |
|---|---|---|
| **可排序编号式**（推荐） | `03 痛点直击-ToFu-服装老板娘：换季断码.png` | 自带顺序、一眼看清 stage/受众/主题 |
| **日期式** | `服装老板娘_20260714_换季断码` | 按时间线找，适合内容日历 |
| **平台式** | `IG-服装老板娘-换季穿搭` | 按平台分，适合多平台分发 |
| **极简式** | `服装老板娘-换季断码` | 最短，内容少时够用 |
> 🔴 文件名里的 `/` 一律转全角 `／`（否则被当路径切分）。

## 🎨 素材（唯一要你自己准备的真东西）

| 素材 | 数量 | 为什么 |
|---|---|---|
| 🔴 **品牌 Logo** | **2 张**：浅底版 + 深底版 | 成品图顶部会叠一条色带放 logo，带色**自适应**（浅底→白带＋深 logo；深底→深带＋白 logo）。PNG 透明底最佳。 |
| **品牌色 hex** ＋ 字体名 | — | 例：`#28c2e7` 青 / `#0f75bc` 蓝 / `#fcb040` 橙。AI 出图照这个走。 |

> ⚠️ **不用准备产品照片** —— 海报图和视频画面**全部由 AI 生成**。
> 唯一要真素材的是 **Logo**，因为 **logo 绝不能让 AI 画**（会画出假 logo）。

## 🧭 定完位，内容角度往哪找？

定位是「你是谁」，内容角度是「每条讲什么」。两个配套工具：

- **6 大人性需求**（见 `启动包/01-《AI品牌定位》启动包/09-6大人性需求-内容角度更有方向.md`）
  给每条内容配一个主需求：**确定感 / 新鲜感 / 重要感 / 爱与连接 / 成长 / 贡献**。
  一批内容 6 种需求轮着来 → 不腻、覆盖全。
- **Hook 谱**：痛点直击 / 对比反差 / 趋势警醒 / 行业揭秘 / 实操演示 / 客户见证 / 算账 ROI / 行动 CTA
  （换成你行业的说法）。
- **NLP 营销技巧**（见 `启动包/04-《AI爆款文案》启动包/08-NLP营销技巧-让文案更有说服力.md`）：挑 1–2 个套进去，别全用。

> 公式：`受众 × 漏斗层(ToFu/MoFu/BoFu) × Hook × 人性需求 × NLP手法 = 一格一条内容`
