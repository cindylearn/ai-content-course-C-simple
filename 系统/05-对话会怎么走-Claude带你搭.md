# 对话会怎么走（装了 skill 之后，你只管答题 + 挑刺）

> **装好 skill 后，你不用背 prompt。** 你只发**一句触发**，Claude 就会按已经装好的框架，一步步**问你、出成品**；
> 你的活只有三样：**① 答它的问题　② 看成品、挑刺　③ 批准了才放量。**
>
> 下面是整场对话会怎么走，每步都给了 SUMA 的答法当参考。

---

## 🟢 你唯一要打的一句（启动）

```
用 ai-marketing-agent-system skill 带我搭我自己的 AI 营销内容系统。
```

→ 之后就是 **Claude 问、你答**。不用再打任何"模板 prompt"。

---

## 第 0 关 · 账号 + 地基

- 🗣️ **Claude 会问**：这 5 个账号有了吗——Claude / Notion / Higgsfield / Suno / GitHub？你的生意做什么？有 logo（浅底+深底）和品牌色吗？
- 🙋 **你答（SUMA 例子）**：卖「AI 内容营销课程」；品牌色 #28c2e7 青 / #0f75bc 蓝 / #fcb040 橙；logo 两版都有。
- 👉 缺账号，Claude 会给你注册链接（一次过装好，之后不用再装）。

## 第 1 关 · 品牌地基（它一题一题问，你一题一题答）

- 🗣️ **Claude 会问**：品牌名？主打？目标受众有哪些？每个受众最痛什么？有什么绝不能写的红线？
- 🙋 **你答（SUMA 例子）**：
  - 品牌名：SUMA College 特别课程【AI内容营销系统 + 线下实操课】
  - 受众：服装店老板娘 / 美妆美容院老板娘 / 内容创作者 / 补习老师
  - 红线：不写价格、不写"免费"、不写具体名额数字（用「名额有限」）、人物一律马来西亚本地人
- 👉 Claude 写进 memory，以后每次生产**自动遵守**，你不用再重复说。

## 第 2 关 · 内容矩阵（它排，你调）

- 🗣️ **Claude 会做**：帮你把 `受众 × 漏斗(ToFu/MoFu/BoFu) × 8 个钩子` 排成一张矩阵，并在 Notion 建好表。
- 🙋 **你做**：看它排得对不对，删几格 / 加几格。

## 第 3 关 · 写第一条（它写，你挑刺）

- 🗣️ **Claude 会做**：挑一格（例：服装店老板娘 × ToFu × 痛点直击），按分层框架写出 ① 标题角度 ② 文案 ③ 配图 prompt。
- 🙋 **你看 → 挑刺（SUMA 例子）**：「太硬卖了，第二段就推课。先共情她『忙到没时间』，第三段才带 AI，语气像朋友。」
- ✅ **满意才放行**：你说「标 Proceed to Generate」，它才进生产。

## 第 4 关 · 出第一张图（它出，你审）

- 🗣️ **Claude 会做**：出图 → 自己先审一遍 → 给你看（**不急着上传**）。
- 🙋 **你挑刺（SUMA 常见）**：「模特是外国脸，换本地人」/「标题撞 logo，加顶部留白带」/「#28c2e7 被印进图里，改用文字描述（青色）」。
- ✅ OK → 它合 logo（顶部 header 带、色自适应）→ 上传 Notion 验证渲染 → 存 Drive。

## 第 5 关 · 出第一支视频（它做，你在关键点点头）

- 🗣️ **Claude 会做**：写 8 分镜脚本 → 出人物参考图锁脸（SUMA 用学姐「陈晓晴」）→ 先跑 **1 镜（480p）验证**给你看。
- 🙋 **你**：1 镜 OK → 说「跑完 2–8，配 Suno 背景乐，拼成完整版」。中文字幕你后期自己加。

## 第 6 关 · 放量 + 记录

- 🙋 **你说**：这条跑通了，套到其他格子批量做——**只做我标了 Proceed 的**。
- 🗣️ **Claude 会做**：批量生产、成品放 Need Review 等你审、push 到你的 GitHub 记录。

---

## 你真正要记住的，不是 prompt，是 3 个判断

1. **一次一步**：看到这步的成品，再往下一步。
2. **不满意就挑刺**：具体到"哪根刺"（太硬卖 / 外国脸 / 色号印进图），别只说"不好"。
3. **你点头才算数**：花 credit / 放量前，标了 `Proceed` 才生产——这是防乱花钱的闸门。
