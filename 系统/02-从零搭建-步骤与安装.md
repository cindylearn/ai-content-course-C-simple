# 从零搭建这套系统 — 步骤 · 装什么 · 连什么

> 你问「我一开始怎么 build 到这么全面的系统？」——这份就是**按顺序**的搭建路线图。
> 先看全景对照表，再照 6 个阶段一步步走。

---

## 全景对照表（每个 part 用什么 · 装什么 · 连什么）

| Part（系统的一块） | 用的工具 | 要装 | 要连（开账号/授权） |
|------|---------|------|------|
| 🤖 编排大脑 | **Claude / Claude Code** | Claude Code | 登录 Claude 账号 |
| 🧠 内容大脑 | **Notion**（免费版） | 浏览器即可 | Notion 账号 + 建一个 database；连给 Claude |
| ✍️ AI 文案 | **Claude** | 同上 | 同上 |
| 🎨 AI 图像 | **Higgsfield** | （MCP/网页） | Higgsfield 账号 + 连给 Claude(MCP) |
| 🤖 AI 数字人 / 🎬 AI 影片 | **Higgsfield (Seedance)** | 同上 | 同上 |
| 🎵 AI 音乐 | **Suno** | 网页 | Suno 账号 |
| 🖼️ Logo 合成 / 视频拼接 | **Python** | `python3` + `pip install pillow imageio-ffmpeg` | — |
| 🗂️ 版本记录 | **GitHub** | `git` | GitHub 账号 + 建 repo + token |
| 🗄️ 成品归档（可选） | **Google Drive** | `rclone` | Drive 账号授权 rclone |

> **最小起步只需要 4 个账号：Claude · Notion · Higgsfield · Suno。** GitHub/Drive 是「记录」和「归档」，可以晚点加。

---

## 阶段 1 · 装工具 & 开账号（Setup）

**开账号（connect）：**
1. **Claude**（Claude Code）— 系统的大脑/编排。
2. **Notion**（免费）— 建一个新的 **database**（不是普通页面）。
3. **Higgsfield** — 出图 + 出视频 + 数字人。
4. **Suno** — 背景音乐。
5.（可选）**GitHub** — 建一个 **private repo** 做记录；生成一个 access token。
6.（可选）**Google Drive** — 成品归档。

**装本地工具（一次性）：**
```bash
# 图片合 logo + 视频拼接
pip install pillow imageio-ffmpeg
# 记录用
git --version        # 没有就装 git
# 归档用（可选）
brew install rclone  # 然后 rclone config 授权你的 Google Drive
```

**准备素材：** 品牌 logo（浅底版 + 深底版各一张）· 品牌色/字体 · CTA 落地页链接 · 合规红线清单。

---

## 阶段 2 · 建大脑（Notion 内容矩阵）

1. Notion 新建 database，加这些列：
   `目标受众 · 领域 · 行业 · 漏斗层(ToFu/MoFu/BoFu) · Hook · 人性需求(6大需求) · NLP手法 · 媒体(图文/视频) · 内容角度(标题) · 文案 · Higgsfield Prompt · 工具标签 · 成品 · Status`
   > 「人性需求」戳哪个（确定感/新鲜感/重要感/爱与连接/成长/贡献，见 09）、「NLP手法」用哪个（见 08）——先标了，产出就有方向。
2. **Status** 用这几个值（这就是流水线开关）：
   `Need Review Idea → Proceed to Generate → Need Review Poster/Video → Checked → Posted`
3. **把 Notion 连给 Claude**：
   - 起步：用官方 Notion connector（授权即可）。
   - 进阶自动化：用 Notion 内部 API（`token_v2` cookie），能批量枚举/批量写/建页。

---

## 阶段 3 · 填内容矩阵（立项，用 AI 文案）

1. 列出你的 **受众 × 漏斗层 × Hook** → 每格一行（受众数 × 8 ≈ 内容总数）。
2. 让 **Claude** 按分层框架 + 你的合规护栏 + persona，逐行写 `内容角度 / 文案 / Higgsfield Prompt`。
3. 想法审好 → Status 标 **`Proceed to Generate`**（批准了才进生产）。

---

## 阶段 4 · 建图文生产线

```
场景 Prompt →(Higgsfield 出图)→ Python 合 logo →(上传 Notion)→ Drive 归档
```
1. 写场景化 prompt（3:4 竖版、本地人物、无乱码、留 logo 位）。
2. **Higgsfield** 出图 → **一定 Read 图人工审**。
3. **Python(PIL)** 顶部叠纯色 header 带合成真 logo（色自适应）。
4. 上传 Notion 成品区（图片附件归 image 块自己 + 验证渲染）→ 改 Status。
5. **Drive** 归档：`[Stage]/[编号受众]/文件名.png`。
> 细节见 `启动包/02-《AI产品图》启动包/README.md`。

---

## 阶段 5 · 建视频生产线

```
8分镜脚本 →(出人物参考图锁脸)→ Seedance 逐镜 →(Suno 配乐)→ 拼接完整版
```
1. 8 分镜脚本进 Notion 表（一 shot 一 row：画面 · 台词 · prompt · 平台）。
2. **Higgsfield** 出一张人物参考图 → 每镜带 `image_references` 锁人物 → Seedance 逐镜（9:16 + 台词音频）。
3. **Suno** 出背景乐；**imageio_ffmpeg** 抽帧验证 + 拼成完整版；中文字幕后期加。
4. 上传 Notion 对应 row + 完整版 → Drive `Video 素材/[编号]/`。
> 细节见 `启动包/05-《AI数字分身》启动包/README.md` · `启动包/03-《AI口播短视频》启动包/README.md` · `启动包/07-《AI品牌配乐》启动包/README.md`。

---

## 阶段 6 · 自动化 & 记录

1. **自动化**：用 Claude 写脚本走 Notion 内部 API（`queryCollection` 枚举、`saveTransactions` 批量写）；大批量改写用**多 agent 并行** + 一份共享标准。
2. **状态驱动**：只生产 `Proceed to Generate` 的；成品进 `Need Review` 等人审。
3. **记录到 GitHub**：把 skill / SOP / memory `git push` 到你的 private repo，做版本记录。

---

## 一句话路线

> **装 4 个账号(Claude/Notion/Higgsfield/Suno) → 建 Notion 矩阵 → 用 Claude 填内容 → 搭图文线 → 搭视频线 → 接自动化 + 推 GitHub 记录。**
>
> 先跑通「1 个受众 × 1 层 × 1 条内容」，再复制成一百条。**One before many。**
