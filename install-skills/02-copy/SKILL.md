## 一、三层，每层卖不同的东西（先判这条是哪层）
- **ToFu 觉醒**：戳痛点/趋势，让人「对，就是我」——**不硬卖、不放 offer/价**。长度 **≈100–160 字（6–7 行）**。
- **MoFu 考虑**：拆方法 / 给证据 / 讲 USP，证明「这方法行」。长度 **≈150–280 字**（更长更实）。
- **BoFu 决策**：算账 ROI + 紧迫 + offer。长度 **≈120–200 字**。
> 🔴 长度只是下限参考——**宁可丰满，别缩成 1–2 句 caption**。

## 二、骨架（照着逐行填满，缺中段=没填完，重写）
- **ToFu**：`【钩子标题】emoji` → 戳痛点(情绪共鸣) → A vs B 对比/后果 → **差距不在X，而在Y** → **好消息：一套 AI 工具一个人就能做（带出工具组：文案/图/视频/数字人/PPT/音乐）** → `日期·完整品牌名·名额有限 👉链接` → `#3–5标签`
- **MoFu**：钩子(拆解/演示/学员故事) → 方法第一步…第二步(或 before→after) → 关键是把几个串成一条流水线、一个人一条龙 → `完整品牌名 👉链接`
- **BoFu**：算账(一年花多少外包/时间) → 学会自己做省下都是你的/当天出成品 → 工具一次学会·方法终身 → `日期·完整品牌名·名额有限 👉链接`

## 三、大师（每条至少 2–3 个，说得出用了谁）
🔴 **Blair Warren 为处境正名**(灵魂,几乎每条:「不是你不够X,是Y」) · **LF8** 戳本能(怕淘汰) · **AIDA/PAS**(PAS→ToFu·AIDA→BoFu) 当骨架 · **Cialdini**(社会认同=见证/稀缺=名额有限) · **Belfort** Before→After 卖 ROI · **Duarte** 客户是英雄你是向导。

## 四、护栏（合规红线，写死）
🔴 不写价格 / 不写「免费」/ 不写具体名额数字（紧迫用「名额有限」）· **完整品牌名一字不差**(每条 1 次,常放 CTA) · **卖方法论不卖工具** · **每条带出工具组** · 无口头禅、要多要实 · **换行用真换行**(别写 `\n`/`nn`)。

## 五、输出 = 进 Notion「文案」列的完整全文
🔴🔴 照 §二 骨架逐行填满、丰满(参照 `04-文案框架` §五 示范)、**不是 2 句 caption**；写完自检：**用了哪 2–3 个大师 + 哪个钩子 + 戳哪个人性需求？说不出＝重写。** 叠 `08-NLP` 的手法。
