# 怎么安装 + 怎么开始

这个 `install-skills/` 里有 **7 个 skill（01-07）**，每个都能单独调用。
> 🔴 **`00-setup`（守门人）在仓库根目录单独放**——**课前先装它**（链接 `…/tree/main/00-setup`），它带你打地基；`install-skills`（01-07）**上课当天再装**。

## 一、装
- **课前**：装根目录的 `00-setup`（跟 Claude 说「帮我从这个链接装这个 skill」/ 或下载后把 `00-setup` 拖进 `~/.claude/skills/`）。
- **课上**：在这个文件夹打开终端 → `bash 一键安装.sh`（把 01-07 全装进 `~/.claude/skills/`）；或把 7 个文件夹逐个拖进去。

## 二、怎么开始（触发 · 两种方式）

🔴 **第一次 / 新业务 → 先触发 `00 守门人`**（课前已装），它会逼你先建好 Notion 地基（dashboard/folder/分类）再放行。
