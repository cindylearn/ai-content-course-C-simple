## 这个包在干嘛
帮用户把「品牌地基」一次性定下来，写进 memory + 业务 md。之后所有 part（出图/文案/视频/数字人/PPT/配乐）都先读这份地基才动手。地基喂对了，后面每一步 AI 才准。**别急着生产，方向先对。**

## 🔴🔴 先搞清「这个业务到底怎么赚钱」（决定所有内容的角度）
问清 **它卖什么、怎么交付**，因为不同商业模式内容角度完全相反：
- **代运营 / agency**（帮客户做营销 = done-for-you）→ 角度「你太忙/做不好，**交给我们**」，绝不说「给你一套系统自己搭」。
- **卖产品 / 电商** → 角度「这产品解决你什么问题」。
- **服务店**（美容/补习/餐饮，到店消费）→ 角度「来店体验/效果/口碑」。
- **教/卖方法**（像 SUMA）→ 角度「教你自己会做」。
🔴 **内容角度必须从『这个业务自己怎么赚钱』长出来**，🔴🔴 **绝不 import SUMA 的角度/产品/卖点**（SUMA 卖「教你自己搭 AI 系统」——只有同样卖方法的业务才这么说，别的业务照搬就是烂成品）。

## 🔴 八二法则：为什么这一步排第一
80% 的结果由最初 20%（方向/地基）决定。品牌定位＝用户的「原始资料库」，喂错了，之后出的每张图、每条文案、每支视频都是**精美但没用**。问完写进 memory，才放行去做内容。

## 品牌地基 · 7 题（一题一题带用户过，每题先给范例再问，问完复述确认）
1. **一句话说清你的生意/品牌做什么**。例：SUMA 是「教人用 AI 做营销内容的课程」。
2. **品牌名 / 主打产品 / CTA 链接**——内容最后把人导去哪（报名页／WhatsApp／官网）。
3. **目标受众 3–5 个**。例：内容创作者、服装店老板娘、美妆美容院老板娘、补习老师。想不出→帮他 brainstorm。
4. **每个受众的 persona：气质 / 会在什么场景出现**。🔴 人物一律**当地人**（例：SUMA 全部马来西亚本地人，含手机屏幕里/模特/背景）。
5. **合规红线：哪些词不能用**。例：不写价格、不写「免费」、不写具体名额数字（用「名额有限」）、品牌名要完整。🔴 写下来 AI 就不会替用户闯祸。
6. **主走哪条内容路线**（🔴 核心分叉，见下）。
7. **成品命名格式**（选一个之后全部统一，见下）。

## 🔴 第 6 题：两条内容路线（必须先选定一条，否则角度飘）
- **① 个人 IP 内容**：做人设/影响力，靠观点+持续输出建信任 → 文案=个人IP文案、视频=数字人口播讲观点。
- **② 卖产品/服务**：直接推货成交 → 文案=卖货文案、视频=产品/服务广告。

## 🔴 第 7 题：命名格式（选一个，之后固定）
- **可排序编号式（推荐）**：`[编号] [钩子]-[Stage]-[受众]：[标题]`。例：`03 痛点直击-ToFu-服装老板娘：换季断码.png`
- 日期式 `[受众]_[YYYYMMDD]_[标题]` · 平台式 `[平台]-[受众]-[标题]` · 极简式 `[受众]-[标题]`
🔴 文件名里 `/` 一律转全角 `／`（否则被当路径切）。

## 素材（唯一要用户自己准备的）
- 🔴 **品牌 Logo 2 张**：浅底版+深底版（PNG 透明底最佳）。🔴 logo 绝不让 AI 画（会出假 logo），成品后期合成。
- **品牌色 hex + 字体名**。例：`#28c2e7` 青 / `#0f75bc` 蓝 / `#fcb040` 橙。
- ⚠️ 不用准备产品照片——画面全 AI 生成。

## 定完位，内容角度往哪找：6 大人性需求
每条内容配一个主需求，一批 6 种轮着来 → 不腻、覆盖全：
- **确定感**（安全不出错）→ 教程/清单/对比测评　**新鲜感**（新奇）→ 反直觉/趋势/幕后　**重要感**（被看见）→ 身份认同/专属限量
- **爱与连接**（归属）→ 社群/共鸣故事　**成长**（变强）→ 转变前后/技能拆解　**贡献**（利他）→ 使命/帮助他人
- 搭配：确定感/成长→最搭 MoFu；新鲜感/重要感→最搭 ToFu；爱与连接/贡献→全漏斗，尤其 BoFu 临门。

## 组装公式
`受众 × 漏斗层(ToFu/MoFu/BoFu) × Hook × 人性需求 = 一格一条内容`。Hook 谱：痛点直击/对比反差/趋势警醒/行业揭秘/实操演示/客户见证/算账ROI/行动CTA（换成用户行业说法）。

## 收尾：写进 memory + 业务 md
7 题问完 → 整理成品牌地基（一句话生意/CTA/受众+persona/红线/内容路线/命名格式/品牌色+logo）写进 memory + 业务 md。🔴 之后每次开工先读它。详见本文件夹 `07-TEMPLATE` + `09-6大人性需求`。

## 🧱 品牌地基 · 7 题（教练会一题一题问你）

> 课前先想好，当天就不卡。写成一个文件存好。

**① 一句话说清你的生意 / 品牌做什么**
> 例：SUMA 是「教人用 AI 做营销内容的课程」。

**② 品牌名 / 主打产品 / CTA 链接**
> 内容最后要把人导去哪里（报名页 / WhatsApp / 官网）。

**③ 目标受众 3–5 个**
> 例（SUMA）：内容创作者、服装店老板娘、美妆美容院老板娘、补习老师…
> 👉 想不出来？让教练帮你 brainstorm。

**④ 每个受众的 persona：气质 / 会在什么场景出现**
> 之后 AI 出的人物就照这个。一两句就行，不用写很细。
> 🔴 人物设定要**贴你当地**（例：SUMA 一律马来西亚本地人）。

**⑤ 你的合规红线：哪些词不能用、不能写什么**
> 例（SUMA）：不写课程价格、不写「免费」、不写具体名额数字（用「名额有限」）、品牌名要写完整。
> 👉 这条是**护栏** —— 写下来，AI 就不会替你闯祸。

**⑥ 你主走哪条路线？**（🔴 核心分叉，决定之后每个 AI Agent 的角度）
| 路线 | 是什么 | 之后的 AI 会怎么跑 |
|---|---|---|
| **① 个人 IP 内容** | 做你自己的影响力 / 人设，靠观点 + 持续输出吸粉、建立信任 | 文案 ＝ 个人 IP 文案 Agent · 影片 ＝ 个人 IP / AI 数字人口播 |
| **② 卖产品 / 服务** | 直接推你的货 / 服务成交 | 文案 ＝ 卖货文案 Agent · 影片 ＝ 产品 / 服务影片广告 |
> 两条都能做，但**每条内容先选定一条**。

**⑦ 成品命名格式**（选一个，之后全部统一用）
| 格式 | 例子 | 好处 |
|---|---|---|
| **可排序编号式**（推荐） | `03 痛点直击-ToFu-服装老板娘：换季断码.png` | 自带顺序、一眼看清 stage/受众/主题 |
| **日期式** | `服装老板娘_20260714_换季断码` | 按时间线找，适合内容日历 |
| **平台式** | `IG-服装老板娘-换季穿搭` | 按平台分，适合多平台分发 |
| **极简式** | `服装老板娘-换季断码` | 最短，内容少时够用 |
> 🔴 文件名里的 `/` 一律转全角 `／`（否则被当路径切分）。

## 📄 09-6大人性需求-内容角度更有方向.md

# 6 大人性需求 —— 让内容角度更有方向（通用，任何行业都能用）

> 🔴 **两条底盘规则（跟 SKILL.md / 本包 README 一致，做这一步也守）：** ① **交互**：问用户任何选择题 → 一题一题、一题一张可点选项卡（每题≤4 个+「其他」）、**绝不一次全倒**。② **底层三层**：出的每样东西都套 说服（大师：**Blair Warren 为处境正名**/自我说服/LF8/AIDA·PAS/Cialdini/Duarte）+ 语言（NLP）+ 动机（6 大人性需求），**说得出「用了哪个大师 + 哪个 NLP + 戳哪个需求」，说不出＝重来。**


> 人做任何决定，背后都在满足 **6 种深层需求**（Tony Robbins 框架）。
> 内容戳中哪一个需求，就更容易被"戳到心里"。
> 这份帮你解决"**不知道下一条内容从什么角度切**"——每条内容先问：**它满足哪个需求？**
>
> 用法：给内容矩阵的每一格**配一个主需求**；一批内容里 6 种需求**轮着来**，受众不腻、覆盖更全。

---

## 🧭 6 大需求速查

| 需求 | 人在追求什么 | 什么内容戳中它 | Hook 例子（换成你的行业）|
|---|---|---|---|
| **1. 确定感 Certainty** | 安全、可控、不出错、不被骗 | 保证、证据、步骤清单、对比测评、"怎么做"教程、退款/售后 | 「照这 3 步做，不会翻车」 |
| **2. 新鲜感 Variety** | 新奇、惊喜、变化、刺激 | 反直觉、新趋势、"你没想到的"、幕后、挑战、盲测 | 「99% 的人不知道的做法」 |
| **3. 重要感 Significance** | 被看见、独特、有地位、被需要 | 身份认同、专属/限量、成就展示、"你值得"、专家/VIP 感 | 「只有认真的人才配拥有」 |
| **4. 爱与连接 Love & Connection** | 归属、被理解、被爱、同类 | 社群、共鸣故事、"我们"、幕后温情、客户关系、身份共同体 | 「原来不只我这样」 |
| **5. 成长 Growth** | 变强、学会、进步、突破 | 教育、转变前后、"升级你的…"、技能拆解、学员进步 | 「30 天从 0 到能自己做」 |
| **6. 贡献 Contribution** | 利他、有意义、影响别人、留下价值 | 使命、帮助他人、回馈、"让更多人…"、公益、传承 | 「学会了，你也能帮到别人」 |

## 💡 怎么把需求变成内容角度（每个需求 3 个角度）

**1. 确定感**：① 手把手教程（照做就行）② 对比/测评（帮你选对）③ 常见坑清单（避雷）
**2. 新鲜感**：① 行业揭秘/内幕 ② 反常识观点 ③ 新工具/新趋势抢先看
**3. 重要感**：① 学员/客户高光时刻 ② 专属身份（"XX 圈内人"）③ 稀缺/限量
**4. 爱与连接**：① 真实故事引共鸣 ② 建立社群/圈子 ③ 幕后/创始人真心话
**5. 成长**：① 转变前后对比 ② 技能拆解（可学会）③ 学习路线图
**6. 贡献**：① 帮助他人的故事 ② "学会去帮别人" ③ 使命/为什么做这件事

## 🔗 怎么跟你的内容矩阵结合

原来的矩阵是 `受众 × 漏斗层 × Hook`。现在**再加一维：人性需求**。

```
受众 × 漏斗层 × Hook × 人性需求 = 一格内容的完整方向
```

- **确定感 / 成长** → 最搭 **MoFu**（讲方法、给证据、教怎么做）
- **新鲜感 / 重要感** → 最搭 **ToFu**（戳注意力、制造好奇/身份）
- **爱与连接 / 贡献** → 全漏斗都能用，尤其做**品牌温度**和**BoFu 临门一脚**

> 小技巧：一个受众的一批内容，**6 种需求各来一两条**，就自然有了节奏和多样性——不会全是硬核教程，也不会全是煽情故事。
