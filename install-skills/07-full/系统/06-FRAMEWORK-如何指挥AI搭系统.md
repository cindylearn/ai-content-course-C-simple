# 如何「指挥 AI」搭一套系统 — 协作框架

> 这份不是教工具，是教**方法**：怎么跟 AI（Claude）一起，从零把一套内容系统做出来。
> 内容全部是从「Cindy 实际怎么带 Claude 做完这个项目」逆向拆解出来的——你照这套节奏，
> 也能指挥 AI 帮你搭你自己的系统。
>
> **核心心法：AI 是执行力，你是判断力。** 你负责品味、方向、决策；AI 负责体力、批量、记忆。

---

## 阶段 0 · 先给 AI「地基」（Ground Truth）

动手前，先让 AI 知道**你是谁、你的生意、你的规矩**。

- **建持久记忆**：把品牌资料、合规红线、目标、人物档案写成 memory / SOP，让 AI 每次开工先读。
- **给参照物**：不用凭空描述——「学习我另一个 project 的文案手法」+ 丢过去的 Google Doc / 链接。
- **实战例子**：> 「你学习我另一个 project【UK Diploma】的文案写作手法…」→ AI 有了范本，输出立刻对味。

**心法：** AI 不是从零猜你，是从你给的地基上长。地基越厚，返工越少。

---

## 6 个协作模式（Pattern）

### Pattern 1 · 从「一件具体小事」切入，别一上来就「帮我搭系统」
你不是说「帮我做一套广告系统」，你是从**修错误链接**这种具体任务开始。
- 具体任务 → 暴露出整条流水线 → 再顺势放大。
> 实战：从「为什么全部 link 都是错的」开始，一路长成整套图文+视频生产线。

**心法：** 小任务是系统的入口。做通一条，再复制成一百条。

### Pattern 2 · 用「信息本来的样子」喂，别用嘴描述
能截图就截图，能指就指，能给例子就给例子。
- 截图：Drive 结构、Notion 表、图裂的报错 → 直接甩给 AI。
- 指现实：> 「这是我的 drive，你要开 folder，然后 upload」
- 给模板：> 「用这个做例子，我希望每一个的 table 都那么大」

**心法：** 一张截图 > 三段文字描述。让 AI 看你看到的。

### Pattern 3 · 「看 → 挑刺 → 纠正」的迭代循环
AI 出东西，你**看成品、挑具体的刺、给具体的改法**，AI 修 + 记进规则不再犯。
- > 「这个图很烂，一点都不吸引人」→ 重做
- > 「服装店不是裁缝店」→ 去掉软尺
- > 「美妆的老板娘应该是漂亮的，不是 aunty」→ 换 persona
- > 「broll 不要 blur 的，不要乱码的」→ 改方向

**心法：** 反馈要**具体到可执行**（不是「不好看」，是「太虚化/像 aunty/有乱码」）。每条纠正都变成一条规则。

### Pattern 4 · 边做边立规矩（SOP 是长出来的）
每次纠正之后：> 「记得更新 sop 和 md」。
- 规则不是一开始想全的，是**踩一个坑记一条**，累积成活的标准。
- 下次 AI 自动遵守，不用你重复讲。

**心法：** 好系统的护栏，都是踩出来的。把坑写下来，就不会掉第二次。

### Pattern 5 · 决策闸门：批准了才花钱
生产前先卡一道：> 「先不生产」/「确认后才生成」/ Status 标 `Proceed to Generate`。
- 用**状态**当开关：只有你点头的才进生产。
- 花 credit / 出量之前，人一定先审。

**心法：** 自动化不等于放手。把「批准」设成一道闸，AI 再快也不乱花钱。

### Pattern 6 · 先验证 1 个，再放量
> 「先执行 1 个 video 验证」→ 好了才批量 32 支。
- 一个跑通（节奏/画质/成本都对）→ 再复制。
- 别照一个没验证的标准批量白做。

**心法：** One before many。验证成本低，返工成本高。

### Pattern 7 · 把「结构性/体力活」丢给 AI，「判断/品味」留给自己
- 丢给 AI：批量改 88 页文案、8 页脚本（多 agent 并行）、跑脚本、拼视频。
- 自己留着：A/B 选择、persona 定调、「我喜欢这个风格」、要不要花这个钱。
> 实战：AI 一次并行改 8 页视频脚本；你只负责最后说「用 A（自然）不用 B（硬塞）」。

**心法：** AI 扩你的手，不代替你的脑。

### Pattern 8 · 「照这个做」——参照锁定
给一个满意的例子，说「我要每个都这样」。
- > 「我喜欢这个风格和画面」→ 后面全照这个走
- > 「文案的 part 用这种画面」→ 锁定视觉方向

**心法：** 与其描述你要什么，不如指一个「就这样」。参照物最省沟通。

---

## 一句话总结这套方法

> **给地基 → 从小事切入 → 用真实信息喂 → 看成品挑具体刺 → 每次纠正立成规则 → 批准了才生产 → 先验一个再放量 → 体力给 AI、判断留自己。**

工具会换，这套「怎么带 AI」的方法不会。学员学会这套节奏，换任何 AI 工具都能自己搭系统。
