# 00 · 无脑安装向导（学员 setup · Mac + Windows）

> 这份给「完全不懂技术」的学员用。你（Claude）按这个带，一次一步、做完再下一步、绝不一次全倒；每步帮他验证。
> 目标：**学员手动只剩「同意运行 + Mac 输一次密码 + 点几下授权」，其余全自动。**

---

## 学员总共只做 3 件事（其余你自动）
1. 装 Claude Code 本体（一条命令）
2. 点授权连 3 个连接器（Notion / Higgsfield / ChatCut）
3. 其余（git/node/python + skill 包）你帮他自动装

---

## Step 0 · 装 Claude Code 本体（唯一要学员先跑的）
- **Mac**：`curl -fsSL https://claude.ai/install.sh | bash`
- **Windows**（PowerShell）：`irm https://claude.ai/install.ps1 | iex`
> 以官网最新命令为准。原生安装，不用先有 Node。

## 🔴 Step 1 · 自动装本地工具（Claude 帮跑，学员不手动下载）
你先侦测系统 + 缺不缺 git/node/python：
- **Mac**（`uname` = Darwin）→ 跑：`bash setup-tools.sh`
- **Windows** → 跑：`powershell -ExecutionPolicy Bypass -File setup-tools.ps1`

学员只需：① 同意你运行命令　②（Mac）中途输一次 Mac 密码。
- Windows 装完 → 提醒学员**重开一次终端**让 PATH 生效再继续。
- 极老 Windows 没 winget → 引导他从 Microsoft Store 装「App Installer」，再重跑。
> **绝不叫学员去官网一个个下载。**

## Step 2 · 连远程连接器（点授权，删不掉的那几下）
这些是**远程连接器**，不用装任何东西，只需登录自己账号点授权：

| 连什么 | 怎么做 |
|---|---|
| **Notion** | 连接器目录搜 "Notion" → 一键授权 |
| **Higgsfield** | 添加自定义连接器 → 贴 URL：`https://mcp.higgsfield.ai/mcp` → 授权 |
| **ChatCut**（要剪辑才连）| 添加自定义连接器 → 贴 URL：`https://api.chatcut.io/api/external-mcp/mcp` → 授权 |

- **Suno**：网页 suno.com 手动做，无需连接。
- **PPT**：Claude 直接做，无需额外工具。
- **GitHub**：学员轨不用（那是生产系统的版本记录）。

## Step 3 · 装 skill 包
- GitHub repo → Code → Download ZIP → 解压 → 文件夹拖进 `~/.claude/skills/`
- 或跑同目录的 `一键安装.sh`。

## Step 4 · 全部验证
逐个 test：Notion 读得到？Higgsfield 出得了图？ChatCut 连上？全绿 = 完成。

---

## 唯一不能自动的（老实告诉学员）
「点授权」要你自己登录自己账号点同意 —— 这是安全设计，任何工具都替不了。**其余我全包。**
